# UAS-ALPRO-DESI
# Program Pengolahan Nilai Mahasiswa

## Deskripsi Program
Program ini dibuat untuk memenuhi Ujian Akhir Semester (UAS) mata kuliah
Algoritma dan Pemrograman Lanjutan dengan dosen pengampu bapak Taofik Muhammad M. Pd
dengan bahasa C ini saya menggunakan untuk mengelola data nilai mahasiswa dengan berisi NIM, nilai tugas, UTS, dan UAS menggunakan array dan fungsi rekursif. sehingga pekerjaan jadi lebih tertata.

---

## Fitur Program
- Input data mahasiswa
- Menampilkan data mahasiswa
- Menambah data mahasiswa
- Mengubah data mahasiswa
- Menghapus data mahasiswa
- Mengolah nilai
- Fungsi rekursif (Faktorial dan Fibonacci)

---

## Cara Compile dan Menjalankan Program

### Compile
gcc nilai_mahasiswa.c -o nilai_mahasiswa
